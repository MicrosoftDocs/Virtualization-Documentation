# HV_X64_REGISTER_SEV_GPA_PAGE

The following data structure is used to define the contents of the Secure AVIC control register (HvX64RegisterSevAvicGpa).

## Syntax

 ```c
typedef union
{
    UINT64 AsUINT64;
    struct
    {
        UINT64 Enabled      : 1;
        UINT64 ReservedZ    : 11;
        UINT64 PageNumber   : 52;
    };

} HV_X64_REGISTER_SEV_GPA_PAGE;
 ```
